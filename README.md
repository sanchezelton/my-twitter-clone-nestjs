# MyTwitterClone

## Description

This project aims to create simple clone of [Twitter/X](https://www.x.com) for the
purposes of demonstrating the [NestJS](https://github.com/nestjs/nest) framework using
its TypeScript starter repository.

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# e2e DB tests (Playwright)
$ yarn run test:e2e:playwright

# test coverage
$ yarn run test:cov
```
